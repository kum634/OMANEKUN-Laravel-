# OMANEKUN-Laravel-
「オーマネ君-整備依頼管理-」のLaravel移植版。

## 概要
 
PHP単体で開発した「オーマネ君-整備依頼管理-」をLaravelに移植したものになります。
ローカル環境のXAMMPで開発したものをAWSにデプロイして動作させております。
 

***デモ***
 
http://omanekun.kum634.com/
 

## 機能
 
- データベースへのデータの挿入、取得、削除。
- キーワード等で任意の条件を指定して検索し、一致したものをデータベースから取得。
- 挿入したデータの日付が今週に当てはまるもののみ、データベースから自動的に取得し表示。
- 取得した値から自動的に作業指示書を作成。
- データベースでユーザー情報を管理（新規登録、ログイン、自動ログイン、パスワード再設定メール、退会機能）。
- CSV入出力機能。
- Bootstrapによるレスポンシブ対応。
